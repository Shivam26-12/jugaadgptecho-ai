# Echo Vision

**Echo Vision** is an intelligent assistive system that combines computer vision, voice interaction, and distance sensing to help users perceive and interpret their surroundings.  
It integrates **YOLOv8** for object detection, a **GPS module** for location tracking, and an **SR04 ultrasonic sensor** for distance measurement ‚Äî all controlled through a Python-based AI system.
